# Using Cursor with Obsidian

[Cursor](https://cursor.sh) is an AI-powered code editor built on VS Code. Since Obsidian vaults are just folders of Markdown files, you can open them in Cursor and use its AI features to edit, search, and reorganize your notes.

## Why use Cursor with Obsidian?

- AI-assisted editing of Markdown files (rewriting, summarizing, restructuring)
- Powerful multi-file search and replace across your vault
- Familiar VS Code interface with Markdown preview
- AI chat that understands the context of your entire vault

## How to get started

1. Install [Cursor](https://cursor.sh)
2. Open your Obsidian vault folder in Cursor (File > Open Folder)
3. Use Cursor's AI features (Cmd+K to edit, Cmd+L to chat) on your Markdown files
