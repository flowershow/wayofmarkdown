# into.md

[into.md](https://into.md/) converts any webpage into clean, LLM-friendly Markdown.

## Usage

Prefix any URL with `into.md/`:

```
into.md/https://zod.dev/v4/changelog
```

That's it. You get back clean Markdown of the page content.

## Why it's useful

- Quick way to grab content from any webpage as Markdown
- Great for feeding web content to LLMs
- No install needed — just prefix the URL
