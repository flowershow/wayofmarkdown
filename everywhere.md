# Markdown is Everywhere 🌍

Markdown started out as the geek underdog that just showed up in text areas on the internet. Now? It's quietly everywhere. Your chat apps speak it. Google Docs speaks it. Every AI speaks it. You've probably been writing markdown for years without anyone telling you.

Here's the map. Each app links to a guide on how to use markdown there: what works, what's quirky, and how to get your text in and out.

## Full markdown support ✅

These get it. Write markdown, get markdown out, no drama.

| App | Notes |
|---|---|
| [Obsidian](markdown-in-obsidian.md) | The markdown PKM champion. It's just files on your disk. |
| [GitHub](markdown-in-github.md) | READMEs, issues, PRs, comments -- markdown everywhere. |
| [ChatGPT, Claude & friends](markdown-in-chatgpt.md) | AI speaks markdown natively. This is a big deal. |
| VS Code / text editors | Obviously. Preview built in. |
| [Reddit](markdown-in-reddit.md) | Full markdown mode in the editor -- links, tables, headings, the lot. |
| Substack, Ghost, Bear, iA Writer, Typora, Zettlr... | The whole writing-app ecosystem. |

## Real support, some quirks 🟡

Markdown's in there, sometimes as a dialect, sometimes hidden behind a setting.

| App | The deal |
|---|---|
| [Google Docs](markdown-in-google-docs.md) | Genuinely good now: paste from markdown, copy as markdown, import/export. You have to switch it on. |
| [WhatsApp](markdown-in-whatsapp.md) | Bold, italic, lists, quotes, code -- with its own accent (single `*asterisks*` for bold). |
| [Discord](markdown-in-discord.md) | Solid subset: headings, lists, code blocks with syntax highlighting, spoilers. |
| [Slack](markdown-in-slack.md) | Speaks "mrkdwn" -- a markdown cousin with opinions. Worth understanding. |
| [Notion](markdown-in-notion.md) | Markdown shortcuts as you type, import and export -- but blocks don't all round-trip. |
| [Telegram](markdown-in-telegram.md) | Bold/italic/code/spoilers via markdown syntax, in apps and bots. |
| [iPad Notes](markdown-in-apple-notes.md) | Import/export arrived in iPadOS 2025. Progress! |

## The holdouts 🙈

Name and shame. We say this with love: it's 2026, folks.

| App | Excuse |
|---|---|
| [Apple Notes (Mac)](markdown-in-apple-notes.md) | Still no native markdown (Apple always have to do things their way). [ProNotes](markdown-in-apple-notes.md) patches it. iPad Notes broke ranks -- hope lives. |
| Microsoft Word | Copilot outputs markdown but Word won't read it. Pandoc to the rescue. |
| Confluence | *Removed* markdown support they used to have. Bold move. |

## Why this list keeps growing

Every tool wants to interoperate, every user wants to learn one format, and every AI is trained on markdown. That's network effects doing their thing, the same reason everyone ends up speaking english at international conferences. Investing in markdown is investing in the thing everything else is converging on.

Which is kind of the whole point of this site 😎 See [why markdown-based](why.md), or jump into [what you can build with it](markdown-based.md).
