# Markdown-based Journals & Daily Notes

> [!note]
> Short version for now -- full guide coming.

One file per day: `2026-07-18.md`. That's the whole system. Obsidian's Daily Notes button does it for you; a template adds your structure (gratitude, tasks, log -- whatever yours is).

**Why markdown for this**: journals are the content you most want to still open in 30 years, which is exactly the plain-text guarantee. They're private by default (files on your disk -- no journaling-app cloud). Search works across decades instantly. And a folder of daily notes is something [AI tools can summarize and mine](kb/ai-editing.md): "what was I working on last March?" is now a real query.

Related: [knowledge bases](markdown-knowledge-bases.md) -- daily notes usually live inside one.
